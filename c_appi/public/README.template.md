Coloca aquí un archivo de Excel llamado `report_template.xlsx` que provenga de una plantilla existente (por ejemplo, descargada de internet).

Recomendaciones de la plantilla:
- Hojas esperadas: `Diario`, `Semanal`, `Mensual`, `Anual` (si no existen, la app las creará al exportar).
- Puedes definir estilos, logotipos, encabezados y fórmulas propias en cada hoja.
- La app añadirá filas con las columnas: `Clave`, `Ventas`, `Importe Total`, `Monto Recibido`, `Cambio Total`, `Ticket Promedio`, `Unidades Totales`.

Ruta de carga en la app: `/report_template.xlsx` (carpeta `public/`).
